# ZZZ Daily Automation

A fun experiment to explore GUI automation

## Flow
Check for dailies done -> launch game -> login -> complete dailies -> collect rewards -> close game

## How to use:
1. Copy `config.example` to `config.json`
2. Fill in the values
3. Run `main.py`
